# 列表-元组-字典-集合区别





| 数据结构         | 是否可变 | 是否重复              | 是否有序 | 定义符合    |
| :--------------- | -------- | --------------------- | -------- | ----------- |
| 列表(list)       | 可变     | 可重复                | 有序     | []          |
| 元组(tuple)      | 不可变   | 可重复                | 有序     | ()          |
| 字典(dictionary) | 可变     | 可重复（key不能重复） | 无序     | {key:value} |
| 集合(set)        | 可变     | 不可重复              | 无序     | {}          |
